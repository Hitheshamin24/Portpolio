## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Hitheshamin24/Portpolio.git
   cd Portpolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 📦 Build & Deployment

### Build for production

```bash
npm run build
```

The output will be in the `dist/` folder.

### Preview the production build locally

```bash
npm run preview
```

### Deploy to Vercel

This project auto-deploys via Vercel on every push to `main`. To deploy manually:

```bash
npm install -g vercel
vercel
```

## 🧩 Use This as Your Own Portfolio

Want a portfolio just like this? You can use this project as a template in minutes!

### Option 1 — GitHub Template (Easiest)
Click the **"Use this template"** button at the top of this repo to instantly create your own copy.

### Option 2 — Fork & Clone

1. **Fork** this repository
2. Clone your fork:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Portpolio.git
   cd Portpolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. **Customize it** — update your name, projects, skills, and contact info inside `src/`
5. Run locally:
   ```bash
   npm run dev
   ```
6. Deploy for free on [Vercel](https://vercel.com/) — just connect your GitHub repo and you're live!
